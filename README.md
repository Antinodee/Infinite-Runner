# 🏃 Infinite Runner

An endless runner game developed using **Unreal Engine 5**, combining **Blueprint** and **C++ scripting** to deliver a fast-paced, procedurally generated gaming experience. Built as a final year B.Tech project at Mahatma Gandhi Institute of Technology, Hyderabad.

---

## 🎮 About the Game

Infinite Runner is a modern take on the endless runner genre. Players navigate through dynamically generated environments — from futuristic cityscapes to surreal otherworldly landscapes — dodging obstacles, collecting power-ups, and chasing high scores. No two runs are ever the same.

---

## ✨ Features

- **Procedural Generation** — Environments, obstacles, and terrain are generated dynamically at runtime, ensuring a unique experience every playthrough
- **Diverse Environments** — Multiple visually distinct landscapes including futuristic cities and surreal realms
- **Responsive Controls** — Precision-tuned mechanics for smooth and fluid player movement
- **Power-ups & Collectibles** — Strategically placed throughout the game to add depth and strategy
- **Scoring & Progression** — Dynamic difficulty scaling, leaderboard-style scoring, and unlockable rewards
- **Sound Design** — Immersive audio with synchronized sound effects, ambient sounds, and adaptive music
- **Optimized Performance** — LOD, occlusion culling, and texture compression for smooth gameplay across hardware

## 🚀 How to Run

1. Clone or download this repository
2. Open the project in **Unreal Engine 5**
3. Select the desired map from the content browser
4. Click **Play** to launch the game from the main menu

---

## 🏗️ System Architecture

The game is built around the following core modules:

- **Procedural Generation** — Algorithmic real-time content creation for terrain and obstacles
- **Rendering & Display** — UE5-powered graphics pipeline with dynamic lighting and effects
- **Performance Optimization** — Memory management, LOD systems, and profiling across devices
- **Sound Design & Integration** — Layered audio system with spatial sound and dynamic music
- **UI/UX Design** — Clean HUD, intuitive menus, and seamless player experience
- **Scoring & Progression** — Score tracking, difficulty scaling, and reward unlocks
